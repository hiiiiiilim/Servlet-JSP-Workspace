Apache : 웹 서버( html 과 같은 정적인 데이터를 처리하는 서버)
		jsp 코드가 포함되어있는 웹페이지를 만들어주는 서버
Tomcat : 웹 서버, 동적 (자바 서블릿을 실행시키고 DB로 연결되어 html과 같은 JSP 파일로 데이터를 받고 전달해주는 역할을함)

JSP: Java Server Page 
 	HTML 와 JAVA 코드를 혼합해서 동적 웹페이지를 생성하는데 도움을 줌
 	JAVA 코드는 <%%> 태그 안에서 작성이 되고 페이지가 실행될때 <%%> 태그안에 작성된 코드는 서블릿 코드로 변환됨
 	
Servlet : java 언어를 사요앻서 만드는 서버측 프로그램임 웹 서비스에서 동작하는 java 클래스 클라이언트의 요청을 받고 요청받은 내용에 대한 응답을 생성해서
		동적인 웹 페이지를 생성하는데 사용됨
